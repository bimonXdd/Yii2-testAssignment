#### Task 1: Write an application in Yii2 called “Group Blog”.
- **Guest** : Can comment and view other posts. Can register an account and log in after registration
- **Admin**(username: admin, password: admin): can create and delete posts, comment. Also has the ability to delete users  but also change users (username, password, email).
- **User:** can create new posts and comment.
-  **Blog**: Has a list of all the posts(taken from DB) with poster and upload time information. Blog supports uploading pictures and videos under 40Mb.
